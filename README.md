# 🤖 Self-Healing CI/CD Pipeline Agent

## 🚀 What I Built
An AI-powered agent that automatically detects and fixes common build errors in CI/CD pipelines.

## ✨ Features
- ✅ **NPM Package Conflicts** - Fixes react/react-dom version mismatches
- ✅ **Python Duplicates** - Cleans up requirements.txt files
- ✅ **Docker Memory Limits** - Adjusts memory limits to prevent crashes
- ✅ **Git Integration** - Automatically creates branches with fixes

## 🎯 How It Works
1. **Scan** - Analyzes build logs and configuration files
2. **Detect** - Identifies errors with confidence scoring
3. **Fix** - Applies appropriate fixes automatically
4. **Commit** - Creates git branches with all fixes
